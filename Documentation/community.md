## :computer:  GitHub repository (code)

The project is available on GitHub at [OpenChemFacts](https://github.com/OpenChemFacts).&#x20;

To contribute:

1. Visit our [GitHub repository](https://github.com/OpenChemFacts) to access Frontend and Backend repo
2. Star the repository to show your support
3. Identify and/or suggest items in Github projects
4. Fork the repository to propose changes&#x20;

## :handshake: Mattermost (discussions)

We use Framateam, an open-source collaboration platform, to facilitate exchanges.&#x20;

The platform is the ideal place to:

* Ask questions about the method and/or platform
* Suggest ideas
* Meet other contributors interested in the project

To join us on Framateam => create your account on [Framateam](https://framateam.org/signup_user_complete/?id=hebhkib9y7g38x97hfideigyro\&md=link\&sbr=su). Once you have access, introduce yourself in the #welcome channel and tell us about your interest in chemical toxicity data.
